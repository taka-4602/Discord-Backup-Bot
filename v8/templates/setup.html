Botのセットアップガイド

前提条件

  • Pythonを使える環境
  • 基本的なファイル操作とネットワーク知識
  • Discord Botアカウントの作成と編集ができること

必要なPythonモジュール

pip install discord.py
pip install flask[async]

Discord Developer Portalでの設定

  1. アプリケーションの作成

    Discord Developer Portalにアクセス

    新しいアプリケーションを作成します

  2. OAuth2の設定

    OAuth2のGeneralからCLIENT IDとCLIENT SECRETを控えます

    Redirectsに以下の形式でFlaskサーバーのURLを入力：

    http://your-domain.com:5000/

    重要：末尾に"/"を入れるのを忘れないでください！

  3. URL Generatorの設定

    SCOPESで"identify"と"guilds.join"を選択

    SELECT REDIRECT URLで先ほど入力したURLを選択

    生成されたURLを控えておきます

  4. Botトークンの取得

    BotセクションからBotトークンをコピーします

ローカルでの作業

v8フォルダをダウンロードし、v8path.pyを編集します：

BOTTOKEN = "Botのトークン"
CLIENT_ID = "OAuth2のClientID"
CLIENT_SECRET = "OAuth2のClientSecret"
REDIRECT_URI = "FlaskサーバーのURL"

Botの起動

python ninV8.py

トラブルシューティング

サーバーに接続できない、500が返される

  • ポートが正しく開放されているか確認
  • グローバルIPから自身のグローバルIPへの接続は不可
  • ローカルテストの場合はローカルIPを使用

/callや/request1でユーザー追加に失敗する

  • Botが対象サーバーに参加しているか確認
  • ユーザーがアプリケーション認証を有効にしているか確認
  • アクセストークンの有効期限（1週間）を確認

ロール付与に失敗する

  • Botにロール付与権限があるか確認
  • Botのロールがターゲットロールより上にあるか確認
